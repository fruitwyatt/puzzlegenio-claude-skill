# Classroom Puzzle Examples

Real-world examples of using the PuzzleGenio skill in Claude Code.

## Vocabulary Review (Crossword)

```
/puzzlegenio I need a crossword puzzle for my 8th grade Spanish vocabulary unit on food
```

> **Recommended:** [Crossword Puzzle Maker](https://puzzlegenio.com/crossword-puzzle-maker)
>
> The AI can auto-generate clues from your word list. Enter your Spanish food vocabulary (e.g., manzana, leche, pan) and the tool creates a crossword with English clues.
>
> **Quick tips:**
> - Use 10-20 words for the best grid density
> - Set difficulty to "medium" for classroom use
> - Download both the puzzle PDF and answer key PDF

## Spelling Practice (Word Search)

```
/puzzlegenio word search with this week's spelling words for 3rd graders
```

> **Recommended:** [Word Search for Kids](https://puzzlegenio.com/word-search-for-kids)
>
> Designed for elementary students with age-appropriate grid sizes and clear fonts.
>
> **Quick tips:**
> - Use a 12x12 grid for 3rd graders (not too big, not too small)
> - Disable diagonal and backward words for easier difficulty
> - Print the worksheet version with lines for writing each word

## Math Warm-Up (Sudoku)

```
/puzzlegenio I need 20 easy sudoku puzzles for a math worksheet packet
```

> **Recommended:** [Sudoku Puzzle Maker](https://puzzlegenio.com/sudoku-puzzle-maker)
>
> Generate up to 10 puzzles per page. Set difficulty to "easy" for warm-up activities.
>
> **Quick tips:**
> - Set "per page" to 4 or 6 for worksheet format
> - Generate 20 puzzles and download as PDF — all puzzles + answer keys in one file
> - Use A4 or Letter paper size depending on your printer

## Party Game (Bingo)

```
/puzzlegenio bingo cards for a science review game, topic: solar system
```

> **Recommended:** [Bingo Card Maker](https://puzzlegenio.com/bingo-card-maker)
>
> AI generates themed word lists. Enter "solar system" and it creates cards with planets, moons, space terms.
>
> **Quick tips:**
> - Generate 30 unique cards for a full classroom
> - Download the caller list PDF separately
> - Each card has a unique arrangement — no two students have the same card
